# Digital Time Capsule

## Overview
Digital Time Capsule is a full-stack web application that allows users to store digital memories such as messages, notes, and media files that can only be accessed after a selected future date. Users can create, manage, lock, unlock, and share their time capsules.

## Features
- User authentication
- Create and manage time capsules
- Schedule future unlock dates
- Public and private capsules
- Media uploads
- Dashboard with capsule statistics
- Public gallery
